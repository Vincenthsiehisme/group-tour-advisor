# Confirmation Log — 待確認規則追蹤表

> 集中追蹤所有散落在 reference 檔案中的 `[雄獅規則－待確認]` 與 `[需業務/法務確認]` 條目。
> 當某條規則被確認後：
> 1. 在本表將狀態改為 `✅ 已確認`，記錄日期與確認內容
> 2. 在原始 reference 檔案中將標注改為 `✅ 已確認（YYYY-MM-DD）：[一句話說明]`

---

## 使用方式

- PM 告知某條規則已確認 → 更新本表 + 原始檔案
- 優先度欄可作為與業務/OP 開確認會議的排程依據
- 「確認對象」欄對應應詢問的角色
