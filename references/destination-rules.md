# Destination Rules — 各目的地業務地雷與產品限制

> 本文件收錄各目的地已知的業務規則、簽證地雷、季節限制、高頻客訴來源。
> 標注說明：[雄獅規則－待確認] / [業界通例] / [需工程確認] / [需與業務確認]
> 教材年份 2017 年，標注 [待確認] 的規則請業務確認現行版本。

---

## 🇦🇺 澳洲 / 🇳🇿 紐西蘭

> 適用產品類型：東澳團（雪梨/墨爾本/布里斯本/凱恩斯/黃金海岸）、紐西蘭南北島團、紐澳合併團

### 簽證地雷

**澳洲（電子簽）**
- 報價預設為中華民國單一國籍身份
- 外國籍護照 或 雙重國籍：加價單點每人 $3,000、雙點以上每人 $5,000 [雄獅規則－待確認]
- 申請人必須持有護照＋身份證字號，且人在台灣才能辦理
- 未成年（未滿18歲）若父母其中一方未同行 → 需父母同意書；雙親皆不同行 → 建議附戶籍謄本
- 護照需有回程6個月以上效期

**紐西蘭（免簽）**
- 台灣護照免簽，條件：中華民國護照正本＋身份證字號＋停留不超過3個月＋護照6個月以上效期
- 外國籍護照不適用免簽，需另行確認辦理資格 [雄獅規則－待確認]

**⚠️ 系統設計風險**
- 報名流程需有「護照國籍」欄位，並在外國籍時自動觸發加價提示或人工審核
- 兒童訂單需有「是否雙親同行」判斷邏輯，缺少時 OP 需人工補件

---

### 季節 / 時序風險

**澳洲**
- 最適旅遊：12–2月（夏季，戶外活動）；6–8月（冬季，潛水）；9–11月（春季，賞鯨）
- 菲利普島神仙企鵝歸巢：天黑後才出現，行程排程需注意當地日落時間，夏季日落較晚
- 藍山國家公園海拔高，早晚溫差大，全年需備厚外套 → 行前說明需特別提醒

**紐西蘭**
- 最適旅遊：12–2月（春夏季，20–30度）
- 第卡波湖薰衣草花季：11月–2月，錯過即無，行程賣點需標示月份限定
- 南島公路拉車時間長（3.5hr–7hr不等），行程設計需預留緩衝，否則景點壓縮易引發客訴
- 福斯冰河 / 法蘭茲約瑟冰河：健行鞋建議直接穿上飛機，放行李箱有被海關開箱風險（鞋底泥土）

---

### 產品規格限制

**澳洲住宿**
- 無三人房型：1房最多4人（2占床＋2不占床），需事先通知 OP 備註，並以飯店當天房況為主
- 標準配置：2人一室（2單床）
- 個人衛生用品需自備（牙刷、牙膏、拖鞋等）

**澳洲餐飲**
- 全程僅送1瓶水（當地水費高）[雄獅規則－待確認]，不比照其他目的地多送
- 素食者：當地素食選項少，餐點份量和菜色有限 → 建議自備素食罐頭，需行前提醒
- 帶泡麵：僅限海鮮口味或素食（葷腥口味入境有動植物管制風險）

**紐西蘭餐飲**
- 自來水可生飲
- 動植物、蛋魚肉、生鮮水果、種籽不可攜帶入境（海關嚴格）

**紐西蘭交通**
- 採用五年內新車（Qualmark 認證）[雄獅規則－待確認]，為雄獅差異化賣點之一
- 台灣飛紐西蘭飛行時間約 11.5hr（不含中停與轉機）

**小費**
- 澳洲：每人每天 USD 10（司機＋領隊＋導遊）
- 紐西蘭：自費可刷卡；小費慣例 [需與業務確認]

### 高頻客訴來源

| 情境 | 說明 | 建議預防 |
|------|------|---------|
| 無尾熊抱抱地點混淆 | 黃金海岸天堂鄉可自行手抱拍照；雪梨地區不行，僅能近距離觀賞 | 行程頁面明確標示「可／不可手抱」；銷售時主動區分 |
| 螢火蟲洞物種混淆 | 布里斯本自然橋 vs 紐西蘭（第阿納洞）是不同物種，皆與台灣螢火蟲不同 | 銷售話術需說明差異，避免旅客認為重複而拒購 |
| 藍山 / 菲利普島衣著 | 海拔高或夜間觀景，明顯低溫，旅客常未備厚外套 | 行前說明書需強制列出「該景點需厚外套」警示 |
| 南島拉車過長 | 基督城→皇后鎮→米佛峽灣路段單程可達 5–7hr，旅客預期落差大 | 行程頁面標示預估拉車時數 |
| 全程僅送1瓶水 | 旅客習慣其他目的地多送水，易產生不滿 | 報名確認信、行前說明均需明確說明 |
| 入境攜帶物品被查扣 | 澳紐海關均對動植物、食品嚴格管制 | 行前說明書列明禁帶清單 |

## 🇨🇳 大陸地區

### 大陸通用規則

**簽證 / 證件**
- 台灣旅客持台胞證免簽入境中國大陸，停留天數由海關決定 [業界通例]
- 文件需備：中華民國護照正本、身分證正本、近期兩吋彩色照、台灣居民口岸簽證申請表 [業界通例]
- 卡式台胞證效期 5 年，建議旅客出發前確認有效期 [雄獅規則－待確認]

**通訊限制**（需行前告知旅客）
- 中國大陸境內 LINE、Facebook 無法使用 [業界通例]
- 車上提供 WiFi 時，Apple 裝置因系統鎖定可能仍無法使用 [雄獅規則－待確認]
- 建議旅客行前自行安裝 VPN 或改用 WeChat 聯絡

**航班**
- 大陸線航班變動性高，需於行前說明會明確告知旅客 [雄獅規則－待確認]

**安全提醒**
- 旅客財物安全需特別提醒，含飯店內亦需注意（扒手問題）[業界通例]
- 茹素旅客需於報名時告知，大陸境內素食供應穩定性因地區不同差異大 [雄獅規則－待確認]

### 華北地區（北京 / 天津 / 山西 / 寧夏）

**產品特性**
- 北京為五環內景點集中型，交通動線可預測，成本控制較穩定
- 山西景點分散，主要車程在 1–4 小時，行程安排需注意體力負荷
- 寧夏銀川景點距市區多為 40 分鐘–3 小時，需確認地接車輛配置

**季節地雷**
- 北京春季（3–5 月）花季旺季，住房緊張，需提早鎖位 [雄獅規則－待確認]
- 寧夏夏季（7–8 月）氣溫高，戶外景點需備遮陽物品

**高頻客訴 / 業務地雷**
- 北京長城景點版本多（八達嶺、居庸關、慕田峪、金山嶺），客人對版本差異有疑問，需行前說明各版本特性
- 購物站（如北京絲綢、玉器等）強迫推銷客訴頻繁，回應需個案處理，必要時與線控確認 [雄獅規則－待確認]

---

### 華中地區（張家界 / 長江三峽 / 西安 / 鄭州）

**產品特性**
- 張家界武陵源景區採大門票制（含環保車），車票已內含，注意說明避免誤解
- 景區纜車費（百龍天梯、索道）不一定含在票價中，需確認行程說明 [雄獅規則－待確認]
- 長江三峽遊船行程時間長（船程 3–4 天），搭配城市接駁複雜，需確認行前文件完整度

**季節地雷**
- 張家界旺季為春（4–5 月）、秋（9–10 月），人潮擁擠，纜車等候時間長需告知旅客

---

### 西南地區（四川 / 廣西 / 貴州 / 雲南）

**客群限制**
- 西南線（西藏、雲南、九寨溝、昆大麗）涉及高海拔，**報名時需確認旅客身體狀況**，有心血管或呼吸道疾病者不適合 [雄獅規則－待確認]
- 黃山（歸安徽，常搭配西南線販售）不適合年長者，行程設計與客群推薦需注意 [雄獅規則－待確認]

**季節地雷**
- 九寨溝旅遊旺季為秋季（9–11 月），搶位壓力大

---

### 雲南省（昆大麗）

**海拔 / 氣候注意**
- 昆明市中心海拔約 1,891 米，麗江古城海拔更高，高原反應風險需告知旅客
- 日溫差大，建議旅客洋蔥式穿法；紫外線強，備遮陽物品

**高頻客訴**
- 麗江古城酒吧街噪音問題，年長客群易有住宿體驗落差，建議選擇古城外圍飯店 [雄獅規則－待確認]

---

### 西藏自治區（高風險）

**入藏證（高風險文件流程）**
- 台灣旅客進入西藏**必須辦理入藏證**，費用通常含於團費 [雄獅規則－待確認]
- 所需文件：臺胞證照片頁影本 + 有效期簽注頁影本；另需註明職業
- **職業限制：公務員、警察、記者、宗教人士不可辦理入藏證** [雄獅規則－待確認]
- 辦證時間：約 **10 個工作天**，收件截止日需倒推
- **65 歲以上旅客須附健康檢查證明** [雄獅規則－待確認]

> ⚠️ P3 設計關鍵：文件截止收件需系統自動計算（出發日 - 10 工作天 - 3 天 buffer），不可依賴 OP 手動推算

**高山症風險**
- 主要景點海拔：拉薩 3,650 米、羊卓庸錯 4,400 米、納木措 4,700 米
- 輕微高山症症狀：頭痛、呼吸急促、胸悶、噁心、耳鳴、失眠
- 心血管 / 呼吸道疾病旅客不適合前往；行前說明會需明確說明高山症風險與應對方式

**適合旅遊季節**
- 4–5 月（旺季開始，人少，可見雪景）
- 6–8 月（旺季，氣候最佳，8 月有雪頓節）
- 9–10 月（淡季，價格便宜，人少）

---

### 華東地區（上海 / 蘇州 / 杭州 / 黃山 / 山東）

**產品特性**
- 江南各景點點對點車程約 2–3 小時，行程排列密度高，需注意旅客體力
- 黃山行程**必須安排山上住宿**，旅客需準備雙肩背包（托運行李無法帶上山）[業界通例]
- 黃山山上飯店自 2012 年起取消免費一次性備品，旅客需自備盥洗用具 [業界通例]
- 上海迪士尼有入園人數 / 票務限制，需確認預訂時間與配額 [需工程確認]

**客群限制**
- 黃山山路以步行為主（約 2.5–3 小時 / 段，有上下坡），不適合年長或體力較差旅客
- 如旅客體力不足可叫滑竿，費用約 350–800 RMB，**非含括費用**，需提前告知

---

### 華南地區（廣東 / 福建 / 海南）

**福建**
- 最適旅遊月份：春季 4–6 月、秋季 9–11 月；夏季多颱風
- 廈門→福州車程約 3.5 小時，行程安排需注意接駁時間

**海南島**
- 三亞定位熱帶海濱，適合各年齡層
- 海口→三亞車程約 4 小時，需確認行程是否含轉場時間

## 🇭🇰 香港特別行政區

**簽證**
- 台灣旅客持台胞證入境香港，申請後可入境兩次，60 天內須入境 [業界通例]
- 停留 30 天（自入境日起算）[業界通例]
- 所需文件：中華民國護照正本（有效期不少於六個月）、台胞證、近期兩吋照、入境許可表

**成團 / 領隊規則（高風險）**
- **10 人即可出團，但不派領隊** [雄獅規則－待確認]
- **需達 15 人才派領隊** [雄獅規則－待確認]
- ⚠️ 此規則影響 P3 名單管理設計：10–14 人出團時，旅客聯絡 / 緊急應變無領隊負責，產品設計需考慮通知機制

**高頻客訴**
- 購物站（珠寶、中藥等）強迫推銷客訴，同大陸線，回應需個案 + 線控確認

## 🇲🇴 澳門特別行政區

**簽證**
- 台灣旅客持有效台灣護照（距回國六個月以上有效期）免簽入境，停留 30 天 [業界通例]
- 所需文件：中華民國護照正本、近期兩吋照、入境許可
- 未滿 18 歲需附父母同意書與戶籍謄本

**成團 / 領隊規則**
- 同港澳通用規則：10 人出團不派領隊，15 人才派 [雄獅規則－待確認]

**產品特性**
- 澳門博弈屬性強，旅客對賭場安排期望需提前管理
- 主要景點集中（大三巴、議事亭前地等），步行可達，適合各年齡層
